Primeros Pasos

¿Qué es CuenlyApp?

CuenlyApp es un sistema inteligente que automatiza la gestión de facturas electrónicas. Conecta con tu correo electrónico, procesa facturas automáticamente y almacena los datos estructurados en MongoDB.

Beneficios principales:
  • Procesamiento automático de facturas desde el correo
  • Extracción inteligente de datos con IA
  • Soporte para PDF y XML (facturas electrónicas)

Automatización

¿Cómo funciona la automatización?

La automatización revisa periódicamente tus correos electrónicos en busca de facturas y las procesa automáticamente sin intervención manual.

Configuración de Intervalo

Define cada cuántos minutos debe ejecutarse el proceso

Revisión de Correos

Busca automáticamente facturas en todas las cuentas configuradas

Controles de automatización
ACTIVO La automatización está funcionando y procesará correos según el intervalo configurado
INACTIVO La automatización está detenida. Los correos no se procesarán automáticamente

Carga Manual de Archivos

Subir archivos PDF

Los archivos PDF se procesan utilizando inteligencia artificial para extraer la información de la factura.

1
Navegar a "Cargar Archivos"

Utiliza el menú de navegación y selecciona "Subir PDF"

2
Seleccionar archivo

Elige el archivo PDF de la factura desde tu computadora

3
Confirmar carga

El sistema procesará el archivo y mostrará los resultados

Subir archivos XML

Los archivos XML (facturas electrónicas) se procesan de forma nativa y suelen tener mayor precisión en la extracción de datos.

Ventaja: Los archivos XML no requieren OpenAI para ser procesados y proporcionan información más estructurada y precisa.

Solución de Problemas

Estado de OpenAI
ACTIVO OpenAI Configurado

El procesamiento de PDF funciona correctamente. Todas las funciones están disponibles.

INACTIVO OpenAI No Disponible

Solo se pueden procesar archivos XML. Los PDF requieren configuración de OpenAI.

Problemas comunes
¿Por qué no se procesan mis correos automáticamente?

Verifica que:

  • La automatización esté activada (botón ACTIVO en el dashboard)
  • Las cuentas de correo estén correctamente configuradas
  • El intervalo de tiempo esté configurado apropiadamente
¿Qué hago si un PDF no se procesa correctamente?

Intenta subir el archivo manualmente cuando OpenAI esté disponible. Si el problema persiste, verifica que el PDF sea legible y contenga información estructurada de factura.
